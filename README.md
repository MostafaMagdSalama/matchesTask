# matchesTask
The Matches Task
## How To Run The Application ?
```bash
1. npm install
2. npm start
```
warning => You have to install nodemon globally or change start script

This command will install the modules 
* express
* mongoose
##DataBase 
* MongoDB Atlas
## End Points
* /list_of_matches
* /matches_date/:date
* /team_name/:name
* /group
## URL And Port
This application is running on port 5000
### URL
```bash
http://localhost:5000/
```
